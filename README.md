# Proect-3-Chatty
Учебный проект: Спецификация требований для веб-приложения Chatty.
В веб-приложение Chatty, решили добавить новую функцию: просмотр видео.
Как системный аналитик, я дополняла функциональные требования для функций карточек для запоминания новых слов, описывала требования к новой функции просмотра видео, рисовала UML Use Case, писала функциональные требования и атрибуты качества, бизнес-правила, глоссарий.
